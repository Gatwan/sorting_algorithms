# Sorting algorithms & Big 0

# Requirements
1. Allowed editors: vi, vim, emacs
2. All your files will be compiled on Ubuntu 20.04 LTS using gcc, using the options -Wall -Werror -Wextra -pedantic -std=gnu89
3. Your code should use the Betty style. It will be checked using betty-style.pl and betty-doc.pl
4. Unless specified otherwise, you are not allowed to use the standard library. Any use of functions like printf, puts, â€¦ is totally forbidden.
5. Use the main.c files to test your functions. 
6. The prototypes of all your functions should be included in your header file called sort.h
